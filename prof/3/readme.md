# Работа с датой и временем

## Таблица преобразования формата даты/времени

| Формат | Значение | Пример |
| :--- | :--- | :--- |
| `%a` | Сокращенное название дня недели | Sun, Mon (en_US); Пн, Вт (ru_RU) |
| `%A` | Полное название дня недели | Sunday, Monday (en_US); понедельник, воскресенье (ru_RU) |
| `%w` | Номер дня недели [0, 6] | 0, 1, …, 6 (0 = воскресенье) |
| `%d` | День месяца [01, 31] | 01, 02, …, 31 |
| `%b` | Сокращенное название месяца | Jan, Feb (en_US); янв, фев (ru_RU) |
| `%B` | Полное название месяца | January, February (en_US); Январь, Февраль (ru_RU) |
| `%m` | Номер месяца [01, 12] | 01, 02, …, 12 |
| `%y` | Год без века [00, 99] | 00, 01, …, 99 |
| `%Y` | Год с веком | 2026; В Linux может быть без ведущих нулей: 1, 2, …, 2026 |
| `%H` | Час (24-часовой формат) [00, 23] | 00, 01, …, 23 |
| `%I` | Час (12-часовой формат) [01, 12] | 01, 02, …, 12 |
| `%p` | AM или PM | AM, PM (en_US) |
| `%M` | Число минут [00, 59] | 00, 01, …, 59 |
| `%S` | Число секунд [00, 59] | 00, 01, …, 59 |
| `%f` | Число микросекунд | 000000, 000001, …, 999999 |
| `%z` | Разница с UTC | +0000, -0400, +1030 |
| `%Z` | Временная зона | UTC, EST, CST |
| `%j` | День года [001, 366] | 001, 002, …, 366 |
| `%U` | Номер недели (с воскресенья) | 00, 01, …, 53 (неделя до 1-го воскр. — нулевая) |
| `%W` | Номер недели (с понедельника) | 00, 01, …, 53 (неделя до 1-го пон. — нулевая) |
| `%c` | Дата и время (локальное) | Mon Jan 26 09:55:00 2026 (en_US); 26.01.2026 09:55:00 (ru_RU) |
| `%x` | Дата (локальная) | 01/26/26 (en_US); 26.01.2026 (ru_RU) |
| `%X` | Время (локальное) | 09:55:00 |
